1. About jJazdy SQL
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
parserJazdy is an open-source free PHP app which allows to generate timetables from mySQL (MariaDB) into HTML.
Templates are based on Smarty engine and can be easily changed.

2. Example use
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
In tpl/ directory there are two example templates (index.html, index_1.html).
Example use of jJazdy is available at http://rozklad-koszalin.tk/rj
This is bus schedule from Koszalin.
